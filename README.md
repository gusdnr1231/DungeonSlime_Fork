- 제작 단위: 팀 (5인)

## 게임 소개
던전의 최하층에서 살고 있던 슬라임이 따스한 햇빛을 받으며 낮잠을 자겠다는 이유로 던전을 탈출하는 이야기
슬라임의 능력인 흡수를 사용해 여러 사물들의 능력을 사용하며 지상으로 도달하는 것이 목표

<br>

## 이 프로젝트에서의 역할
인게임 요소 제작 : 상자, 거미, 스켈레톤, 부서지는 발판 <br>
인게임 요소 기획 : 스테이지의 테마, 새로운 적/보스 <br>
스테이지 제작 : 전체적인 스테이지의 제작을 도맡아함 <br>
테스트 및 버그 수정
